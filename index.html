About

I'm Daniel — a full-stack developer who builds web apps, mobile apps, and e-commerce sites for founders and small teams. I care about code that ships and products that actually get used.

I work across the stack: React / Next.js / Node on the front, Python (Django, FastAPI, Flask) on the back, React Native & Flutter for mobile, plus Shopify, WordPress and Webflow when speed matters more than custom code.

Recent work spans a premium boba brand's landing site, an LPG availability app, an RV park booking platform, and a modern blogging SaaS. Each shipped end-to-end — design, code, deploy.

Based in Tamale, Ghana. I'm curious and driven — I enjoy turning ideas into tangible results that make an impact. I value clear communication, thoughtful execution, and attention to detail, and I'm happiest when a project pushes me beyond my comfort zone. Always exploring new tools and ideas, always refining the craft.
